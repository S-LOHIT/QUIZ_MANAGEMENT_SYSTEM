# Quiz Management System

## Overview
This project automates quiz creation, administration, and evaluation. Teachers can create quizzes with various question types, and students can take timed quizzes online. The system provides instant grading and detailed performance reports.

## Features
- Secure user login and role-based access
- Multiple question types (MCQs, True/False)
- Automated grading and instant results
- Timed quizzes with navigation controls
- Question randomization for fairness
- Detailed analytics and score reports

## Technology Stack
- Frontend: HTML, CSS, JavaScript
- Backend: PHP
- Database: MySQL
- Web Server: Apache

## Installation
1. Clone the repository
2. Set up the MySQL database using provided schema
3. Configure database connection in the PHP files
4. Deploy on Apache server
5. Access via browser and register users

## Usage
- Admin users can create, edit, or delete quizzes and questions
- Students can log in, take quizzes, and view results instantly
